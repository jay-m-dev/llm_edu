# Tokenizer Prototype

A minimal, standalone tokenizer prototype for the LLM simulator with pseudo-embeddings, attention weights, a transformation pipeline, context window mechanics, sampling controls, deterministic replay, token-by-token generation, per-token probability tracking, step attention tracking, replay controls, a basic objective system, failure diagnostics, multi-axis scoring, unlockable parameters, sandbox mode, and hallucination indicators.

## Run
Open `prototype/index.html` in a browser.

## Test
From the repo root:
```
npm test
```
